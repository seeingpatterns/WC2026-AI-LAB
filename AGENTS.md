# AGENTS — 작업 규칙 (WC2026 AI LAB)

## 실행 원칙
- **정적·클라이언트 전용.** 백엔드/빌드 도구 도입 금지. 브라우저에서 바로 열려야 함.
- 데이터는 `assets/data.js` 단일 소스만 수정. 페이지 코드에 수치 하드코딩 금지.
- 공유 로직(`faces.js`, `style.css`)을 재사용. 중복 구현 금지.
- 새 페이지 = 기존 nav 구조 + 공유 에셋을 따른다.

## 검증 (출고 전 필수)
- `<script>` 추출 후 `node --check`로 문법 검증.
- 콘솔 에러 0. 모든 페이지가 file:// 로 단독 동작하는지 확인.
- 수치는 출처(FIFA EFI)와 대조. 모델/추정은 라벨로 구분.

## 데이터 정직성
- ✅REAL = FIFA EFI 실측. 그 외(동선/예측/얼굴) = '모델/재구성'으로 표기.
- 검색에 딸려온 무관 출처 사용 금지(교차검증).

## 안전
- 개인정보/민감정보 임베드 금지. 외부 호출 없음(완전 정적).
