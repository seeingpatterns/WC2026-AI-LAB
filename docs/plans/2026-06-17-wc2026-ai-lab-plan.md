# PLAN — WC2026 AI LAB (2026-06-17)

## 목표
FIFA EFI 실측으로, 사람들이 직접 만지는 공개 전술 사이트. 5개 인터랙티브 탭.

## 결정사항
- 정적 사이트(빌드 X) → GitHub Pages. 데이터 단일소스 `assets/data.js`.
- 디자인: 다크 배경 + 골드 강조(도감 톤) 통일. 공유 `style.css`.
- 모델 = 결정적 휴리스틱(클라이언트). LLM 실시간 호출 없음 → 호스팅 무비용.

## 범위(우선순위)
1. ✅ 스캐폴드 + 공유 에셋 + 랜딩 + 선수 도감
2. 🦋 평행우주 (변수 토글 → 결과 재계산)
3. 👻 유령 토너먼트 (팀 레이팅 → 대진 시뮬)
4. 🔮 예언 엔진 (예언 발행 → 자가채점)
5. 🧬 팀 DNA/상성 (보너스)

## 비목표
- 실시간 라이브 데이터 연동, 로그인, 결제, 베팅.
